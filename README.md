# Student Management System (Python CLI)

A simple command-line based student management system built using Python.
This project demonstrates the use of functions, modules, and variable scope.

---

## 🚀 Features

* Add student with marks
* View all students
* Calculate average marks
* Simple CLI interface

---

## 🧠 Concepts Used

* Functions
* Modules (student_data.py)
* Scope (global vs local variables)
* Lists & Dictionaries
* Input/Output handling

---

## ▶️ How to Run

* python main.py
